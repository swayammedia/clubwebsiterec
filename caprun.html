CAPSULE RUNNER GAME PROTOTYPE

Capsule Runner is a game that makes use of all the basic mechanics of a Endless runner games ( ex: Subway surfers ) made by using only the default assets in the Unity Engine In Capsule Runner ,The player need to reach the Tree at the End without colliding on obstacles in their ways while collecting coins to Complete the level.
